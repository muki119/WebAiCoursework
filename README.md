# WebAiCoursework
Implementing Machine Learning on the web using Tensorflow.js

## Overview of Features
- Load pre-trained models (MobileNet V1, MobileNet V2, Lite MobileNet V2)
- Support for multiple backends (WebGL, CPU)
- Image upload and real-time classification
- Display model metadata (name, version, size)

# Intallation
Requriements:
- Node.js (v22.18.x or later)

1. Clone the repository:
   ```bash
   git clone https://github.com/muki119/WebAiCoursework.git
    ```
2. Navigate to the project directory:
    ```bash
    cd WebAiCoursework
     ```    
3. Install the dependencies:
    ```bash
    npm install
     ```
# Running the Application
1. Start the development server:
    ```bash
    npm start
     ```
2. Open your web browser and go to `http://localhost:8080` to access the application.
